📦 Inventory Manager App

A smart and user-friendly Android application built to manage product inventory efficiently. This app helps users track stock levels, monitor expiry dates, and organize products with an intuitive dashboard interface.

✨ Features
📊 Dashboard with inventory statistics
⏰ Expiry tracking (Expired / Expiring Soon / Safe)
🔍 Search and filter products easily
🗂️ Category-based product management
✏️ Edit product details with a single tap
🗑️ Swipe to delete items quickly
➕ Floating action button to add new products
📉 Low stock monitoring
🎨 Clean and modern Material UI design


🛠️ Tech Stack
Java / Kotlin
Android Studio
SQLite / Room Database
RecyclerView
Material Design Components
📱 App Purpose


This application is designed for small shops, grocery stores, home inventory management, and anyone who wants to keep track of products and expiry dates in a simple way.
